The workflow

Seven modules, one ticker flow

Pick a name in the Screener or the search bar and it carries through every module automatically.

01

Screener

Composite-score ranking across a global basket (US, India, EU, Korea). Buy / Watch / Avoid, with a rotating daily Top Pick.

02

Deep Dive

Interactive candlestick + indicators, fundamentals, an auto bull/bear memo, sector & macro context, live news with summaries, and analyst consensus.

03

Valuation

Country-correct DCF (Damodaran WACC across 12 markets), comps, DDM, and a bear/base/bull football-field vs the current price.

04

Backtester

Full tearsheet with a Plotly equity curve — entry/exit markers, benchmark overlay, trade log, monthly-returns heatmap.

05

Monte Carlo

Animated percentile cone under GBM, GARCH-t, or Ornstein-Uhlenbeck, VaR/CVaR, plus an MC-derived trade setup and cross-layer decision signal.

06

Strategy Library

Every strategy back-tested live on the active ticker, ranked by Sharpe, each with a BUY / HOLD / AVOID verdict. One click loads it into the Backtester.

07

Portfolio Optimizer

Markowitz max-Sharpe over a persistent basket — efficient frontier, correlation heatmap, per-position risk, and live P&L on your logged buys.

Under the hood

Built on a clean, testable core

A thin FastAPI backend turns pure-Python quant layers into JSON; a single vanilla-JS frontend renders them with Plotly. The financial logic has zero web imports — so it's all unit-tested.

For your own use

Run it in three commands

Clone it, install the deps, start the server. No keys, no signup — it fetches live data from Yahoo Finance at request time.

1

Clone & install

Grab the repo and install the Python dependencies into your environment.

2

Start the server

One uvicorn command serves the API and the frontend together.

3

Open the deck

Visit localhost:8000 and start with any ticker — that's it.

# 1 — clone & install
git clone https://github.com/chintan-jikkar/QuantDeck.git
cd QuantDeck
pip install -r requirements.txt

# 2 — run (no API keys needed)
uvicorn api.main:app --port 8000

# 3 — open
→ http://localhost:8000

Not financial advice. QuantDeck is a quantitative-analysis and educational tool. Every signal, valuation, and backtest is a model output.
