Seven modules, one ticker flow

Pick a name in the Screener or the search bar and it carries through every module automatically.

Screener

02

Deep Dive

Interactive candlestick + indicators, fundamentals, an auto bull/bear memo, live macro regime, news with summaries, and analyst consensus. FX/commodities get momentum, vol & RSI.

03

Valuation

Country-correct DCF (Damodaran WACC across 12 markets), comps, DDM, and a bear/base/bull football-field vs the current price.

04

Backtester

Full tearsheet with a Plotly equity curve — entry/exit markers, benchmark overlay, trade log, monthly-returns heatmap.

05

Monte Carlo

Animated percentile cone under GBM, GARCH-t, or Ornstein-Uhlenbeck, VaR/CVaR, plus an MC-derived trade setup and cross-layer decision signal.

Strategy Library

Every strategy back-tested live on the active ticker, ranked by Sharpe, each with a BUY / HOLD / AVOID verdict. One click loads it into the Backtester.

07

Portfolio Optimizer

True Markowitz max-Sharpe (solved, not sampled) over a persistent basket — efficient frontier, correlation heatmap, per-position risk, live P&L.

Make it your own

Three ways to extend it

The layer boundary is the whole point — layers/ and strategies/ never import web code, so adding to them never touches the API or the UI.

Add a strategy

Subclass Strategy, implement one method, register it — the Backtester and Strategy Library pick it up automatically.

# strategies/my_strategy.py
class MyStrategy(Strategy):
    def generate_signals(self, prices_df, **params):
        # 1 long, -1 short, 0 flat
        ...

# strategies/__init__.py
register(MyStrategy())

Add a universe

One entry in config.py — ticker suffix, benchmark ETF, local index for beta, and country for WACC.

# config.py
EQUITY_UNIVERSES["ASX 200"] = {
    "suffix": ".AX", "benchmark": "EWA",
    "market_index": "^AXJO",
    "country": "Australia",
}

Add a country

Valuation inherits it instantly — Damodaran risk-free ticker, equity risk premium, and country risk premium.

# config.py
COUNTRY_RISK["Brazil"] = {
    "rf_ticker": "IRLTLT01BRM156N",
    "erp": 0.0472, "crp": 0.021,
}
For your own use

Run it in three commands

Clone it, install the deps, start the server. No keys, no signup — it fetches live data from Yahoo Finance at request time.

1

Clone & install

Grab the repo and install the Python dependencies into your environment.

2

Start the server

One uvicorn command serves the API and the frontend together.

3

Open the deck

Visit localhost:8000 and start with any ticker — that's it.

# 1 — clone & install
git clone https://github.com/chintan-jikkar/QuantDeck.git
cd QuantDeck
pip install -r requirements.txt

# 2 — run (no API keys needed)
uvicorn api.main:app --port 8000

# 3 — open
→ http://localhost:8000
